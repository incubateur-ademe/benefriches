Les contributions publiques de collectivités sont de plusieurs types :

- ventes de foncier pour équipements et espaces publics à la collectivité,
- participation pour remise d’ouvrage (en concession),
- apport en nature (foncier, etc.),
- subvention d’équilibre (concédant ou régie).

La valeur est saisie par l’utilisateur.

**Bénéficiaire** : aménageur
