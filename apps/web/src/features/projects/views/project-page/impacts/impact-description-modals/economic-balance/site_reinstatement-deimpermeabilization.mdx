Du fait des activités antérieures (ex : usine) ou de l’utilisation antérieure d’un site (ex : centre commercial), les sols ont pu être largement imperméabilisés (voiries, parking, aires de stockage, dalles des bâtiments, etc.).

Aussi le retrait de ces surfaces imperméabilisées peut être nécessaire pour les besoins du projet.

Au cas où l’utilisateur ne renseigne pas de montant de dépenses de désimperméabilisation mais que le bilan des surfaces calculé entre le site et le projet est négatif, Bénéfriches calcule un montant de dépenses sur la base du retour d’expérience documenté par l’ADEME.

Bénéficiaire : aménageur

## Comment est fait le calcul ?

Lorsque le bilan des surfaces imperméabilisées entre le site et le projet est négatif, les dépenses de désimperméabilisation sont calculées en multipliant la surface desimperméabilisée (exprimée en m²) par le montant de dépenses moyen (exprimé en €/m² de surface) pour la désimperméabilisation.
