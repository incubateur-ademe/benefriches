Au-delà de la remise en état, la réalisation du projet nécessite une préparation du foncier (son aménagement) qui comporte plusieurs postes de dépenses :

- les études et honoraires techniques pour concevoir et assurer le suivi de la mise en oeuvre,
- les travaux d’aménagement à proprement parler
- d’autres dépenses d’aménagement (tels que les provisions pour aléas ou charge d’actualisation si l’aménagement se déroule sur une longue période, comme c’est le cas des ZAC par exemple).
