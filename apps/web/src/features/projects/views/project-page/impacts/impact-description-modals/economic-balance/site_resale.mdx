Si le site a été acheté pour pouvoir réaliser le projet (aménagement ou construction), il pourra être cédé sous différentes formes :

- droits à construire aux promoteurs
- vente en l’état futur d’achèvement
- vente directe de terrains aménagés à des particuliers ou entreprises utilisatrices (ex : industriels, etc.).

Tout ou partie du foncier remis en état puis aménagé ou même construit pourra être conservé par l’aménageur pour ses propres besoins.

La valeur est saisie par l’utilisateur.
