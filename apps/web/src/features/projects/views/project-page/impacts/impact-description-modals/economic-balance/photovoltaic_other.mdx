Les autres frais d’installation de la centrale sont la location (cas le plus fréquent) du foncier (location), les frais financiers, les taxes, la rémunération du porteur de projet, les taxes voire les provisions pour aléas.

## Quelles données sont utilisées dans le calcul ?

### Données systémiques agrégées par Bénéfriches :

- Montant de dépenses moyen pour les autres frais d’installation de la centrale PV : 79 €/kWc

### Données du site :

Sans objet

### Données du projet :

_Les données du site peuvent avoir été saisies par l’utilisateur·ice ou avoir été suggérées par Bénéfriches sur la base d’une moyenne ou d’une hypothèse._

- Puissance de l’installation (kWc)

## Comment est fait le calcul ?

Les dépenses d’études et honoraires techniques sont calculées en multipliant la puissance de l’installation (exprimée en kWc) par le montant de dépenses moyen (exprimé en €/kWc) pour ce poste de dépenses issu du retour d’expérience documenté par la Commission de régulation de l’énergie.
