Les subventions publiques (Etat, région, ANAH, ANRU, ADEME, etc.) attribuées pour financer certaines dépenses (ex : fonds vert “recyclage foncier” pour la remise en état), selon le type d’opération, sa localisation géographique ou son inscription dans un programme particulier (ex : quartier prioritaire de la ville).

La valeur est saisie par l’utilisateur.
